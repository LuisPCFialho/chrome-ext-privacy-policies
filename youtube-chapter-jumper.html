YouTube Chapter Jumper - Privacy Policy

Effective date: 2026-04-15

Data Collection

YouTube Chapter Jumper does NOT collect, store, or transmit any personal user data to external servers. All data is stored locally on your device using chrome.storage.

External Communication

The extension makes HTTPS calls exclusively to api.lemonsqueezy.com for Pro license key validation. Only the license key string is transmitted.

Permissions

The extension requests only the permissions necessary for its core functionality. All data processing happens locally in your browser.
