Features

Bookmark Sorter organizes your Chrome bookmarks — sorting folders alphabetically, finding duplicates, merging folders, and cleaning up dead links.

Free

  • Sort all bookmarks alphabetically (A-Z or Z-A)
  • Sort folders first, then bookmarks
  • Sort a single folder or all folders recursively
  • Preview changes before applying
  • Full undo history - restore to any previous snapshot

Pro Features

  • Find and remove duplicate bookmarks
  • Detect dead/broken bookmark links
  • Custom sort rules (by title, URL, or date added)
  • Empty folder cleanup
  • Merge duplicate folders
  • Export bookmark report

All data stays on your device. No tracking, no accounts, no data collection.
