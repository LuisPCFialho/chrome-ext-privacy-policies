Privacy Policy — Note Highlighter

Last updated: April 2026  |  Developer: LuisPCFialho

1. Overview

Note Highlighter is a Chrome extension that highlights text on webpages and saves persistent notes attached to highlights. All processing happens locally in your browser. No personal data is collected or transmitted to the developer.

2. Data Storage

All data (settings, history, preferences) is stored exclusively on your device using chrome.storage. It is never uploaded to external servers.

3. Permissions Explained

  • storage — Saves your highlights and notes locally, keyed to each page URL.
  • activeTab / scripting — Injects highlight markers and note annotations into pages.
  • tabs — Associates notes with the correct page URL.

4. External Services

This extension communicates only with: api.lemonsqueezy.com (license key validation only). No other external connections are made.

5. Data Sharing

No user data is sold, rented, or shared with any third party.

6. Remote Code

This extension does not load or execute any remotely hosted code. All extension logic is bundled in the installed package.
