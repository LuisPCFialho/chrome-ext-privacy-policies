Privacy Policy

ColorPick Pro — Chrome Extension

No personal data collected

1. Data Collection

ColorPick Pro does not collect, transmit, or store any personal data. The extension does not gather browsing history, form inputs, credentials, analytics, or any other personally identifiable information.

2. Data Storage

All user preferences and extension data are stored locally on your device using the chrome.storage API. This data never leaves your browser and is not accessible to the publisher or any third party.

3. External Communication

The extension makes a single outbound HTTPS request to api.lemonsqueezy.com solely for the purpose of license key validation. This request contains only the license key you provide and does not include any personal or browsing data.

4. Permissions Used

The extension requests only the minimum browser permissions necessary for its core functionality. No permissions are used to monitor, collect, or transmit personal data. Specific permissions and their justifications are detailed on the Chrome Web Store listing page.

5. Third-Party Services

  • Lemon Squeezy (privacy policy) — used exclusively for license validation. No personal data is shared with this service beyond the license key.

No other third-party services, analytics platforms, advertising networks, or tracking tools are used.

6. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with a revised effective date. Continued use of the extension after changes are posted constitutes acceptance of the updated policy.
